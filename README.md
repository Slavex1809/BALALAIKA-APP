\# BALALAIKA Platform



Платформа для аренды оборудования.



\## Запуск проекта



1\. Установите зависимости:

npm run install-all



2\. Запустите базу данных:

npm run supabase:start



3\. Запустите веб-приложение:

npm run web



4\. Запустите мобильное приложение:

npm run mobile



\## Доступ к базе данных

\- Панель управления: http://localhost:54322

\- База данных: localhost:54321
